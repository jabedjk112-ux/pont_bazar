# Pont Bazar

Pont Bazar একটি ই-কমার্স অ্যাপ যেখানে হোলসেল প্যান্টসহ বিভিন্ন পণ্য অর্ডার করা যাবে।  
এই প্রোজেক্টটি Flutter দিয়ে তৈরি করা হচ্ছে।

## Features (প্ল্যান)

- হোম স্ক্রিন (সার্চ, ক্যাটাগরি, ডিল)
- প্রোডাক্ট লিস্ট ও ডিটেইল
- কার্ট ও অর্ডার প্লেস
- অ্যাডমিন প্যানেল (পরবর্তীতে)
- AI অ্যাসিস্ট্যান্ট (পরবর্তীতে)
